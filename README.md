# Контактна форма - Тестове завдання

Веб-сторінка з формою для користувача з валідацією та відправкою email.

## Вимоги

- Node.js 18+ 
- npm або yarn

## Встановлення

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Відкрийте браузер на адресі, яку покаже Vite (зазвичай `http://localhost:5173`)

## Збірка для продакшн

```bash
npm run build
```

## Функціонал

- ✅ Поле "Ім'я" (необов'язкове)
- ✅ Поле "Email" (обов'язкове з валідацією)
- ✅ Текстове поле "Повідомлення" (необов'язкове)
- ✅ Валідація email формату
- ✅ Відображення помилок валідації
- ✅ Відправка email на адресу **6weeks.13h@gmail.com**
- ✅ Тема листа: **"6weeks - Форма заповнена"**
- ✅ Тіло листа містить всі дані з форми
- ✅ Сучасний та привабливий дизайн

## Відправка Email

Форма використовує **FormSubmit** - простий сервіс, який працює одразу без налаштування!

Листи автоматично надходять на адресу `6weeks.13h@gmail.com` з темою "6weeks - Форма заповнена".

### Як це працює

FormSubmit - це безкоштовний сервіс, який дозволяє відправляти email з форм без налаштування серверів. Він працює через API запити і не потребує реєстрації або налаштування.

## Технології

- React 18
- TypeScript
- Vite
- FormSubmit (для відправки email)

## Тестування

1. Запустіть проект: `npm run dev`
2. Заповніть форму на сторінці
3. Переконайтеся, що email вказано коректно
4. Натисніть "Відправити"
5. Перевірте пошту **6weeks.13h@gmail.com**

Форма готова до використання одразу після запуску!
